# Vampire Shop

一个展示吸血鬼主题商品的在线商店网站。

## 功能特点

- 响应式设计，适配各种设备
- 3D 商品展示
- 购物车功能
- 优惠码系统
- 社交媒体链接

## 技术栈

- HTML5
- CSS3
- JavaScript
- Three.js (3D 渲染)
- Font Awesome (图标)

## 本地运行

1. 克隆仓库
```bash
git clone https://github.com/你的用户名/vampire-shop.git
```

2. 打开项目文件夹
```bash
cd vampire-shop
```

3. 使用浏览器打开 index.html 文件
